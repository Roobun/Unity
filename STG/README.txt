Вкратце объясню, какие сущности я использовал:
  Player/PlayerView/PlayerController -- здесь реализована логика, связанная с кораблем (передвижение, стрельба)
  GameController/Game -- тут находятся общие вещи для игры: очки, жизни. При переходе на следующий уровень здесь вызывается SaveGameController, который запоминает важные для игры поля.
  LevelController/Level -- через него спаунятся астероиды. Также тут хранятся уникальные для уровня хар-ки: очки до конца уровня, кол-во астероидов. С переходом на новый уровень эти значения увеличиваются на случайную величину.
  AsteroidController -- в нем происходит взаимодействие игрока, астероидов и пуль.
  PoolController -- реализация пула, применяемого для астероидов и пуль.
  SaveGameController -- механика сохраний/загрузок.
  BoundaryController -- выключает объекты, вышедшие за границы видимости.
  UI -- собирает данные с Game и Level для отображения на интерфейсе.
